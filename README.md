# Agentic AI Lab

A progressive series of AI agent assignments built with Python and Ollama (local LLM).

## Assignments

| Day | Assignment | Concept |
|-----|-----------|---------|
| day1 | Rule-Based Agent | Keyword matching, input→decision→action |
| day2 | Tool-Using Agent | Modular tools, function calling |
| day3 | LLM-Based Agent | Natural language decision making |
| day4 | Multi-Step Agent | Task decomposition, planning |

## Setup
- Python 3.9+
- Ollama with llama3.2 model

## Run any assignment
```bash
cd day1  # or day2, day3, day4
python3 agent.py
```
