# Assignment 1: Rule-Based AI Agent

## What it does
A simple agent that takes user input, matches keywords, and performs actions.

## How it works
- Input Handler: reads user query
- Decision Logic: matches keywords (hello, date, calculate)
- Action Execution: runs the correct function

## How to run
```bash
python3 agent.py
```

## Example
- "hello" → greeting
- "date" → today's date
- "calculate 2+3" → 5
