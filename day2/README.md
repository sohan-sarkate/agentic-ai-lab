# Assignment 2: Tool-Using Agent

## What it does
Agent that selects and uses external tools based on user input.

## Tools
- `calculator` - evaluates math expressions
- `get_weather` - returns mocked weather data
- `summarize` - shortens long text

## Files
- `tools.py` - all tool definitions
- `agent.py` - decision logic and input handler

## How to run
```bash
python3 agent.py
```
